# BM TOOLS 商业运营版迭代记录

本文档用于记录 BM TOOLS 面向商业运营版本的功能新增、页面修改、内容删除与体验优化。后续每轮开发都建议继续维护这份文档，便于测试、交付、售后和二次开发。

  - 待处理订单

- 新增本机服务器环境信息展示：

  - PHP 版本

## 后续建议

下一轮建议优先继续完善以下能力：

1. 后台订单详情页：展示订单时间线、支付信息、用户信息、操作日志。
2. 后台充值审核弹窗：支持审核通过、驳回、备注、凭证预览。
3. API Token 重置：用户可以主动刷新 Token，旧 Token 立即失效。
4. PicGo / ShareX 独立教程页：前台公告或帮助中心中增加可公开访问教程。
5. 文件下载统计：为图床和网盘增加访问量、下载量、热门文件排行。
6. 存储健康巡检：定时测试默认存储源是否可写、可删、可访问。
7. 支付演练模式：后台提供模拟支付成功、模拟退款、模拟补单流程，便于运营上线前自测。

### 商业价值

- 对开发者和图床重度用户更友好，符合商业 SaaS 的账号安全预期。
