# BM TOOLS

> 一套基于 PHP 7.4 + MySQL 5.7 的综合型图床、轻量网盘、充值套餐、多对象存储与服务器状态监控系统。  
> 本项目仅用于正规学习研究、正向开发教学、个人站点搭建与合法业务系统开发，不包含也不支持任何违规用途。

## 项目简介

BM TOOLS 是一套面向个人站长、开发者、内容创作者和小团队的轻量云工具平台。系统将图床服务、个人网盘、套餐订阅、余额充值、对象存储接入、公告通知、开放 API、服务器状态展示和后台运营管理整合到同一个 PHP 项目中，适合部署在宝塔面板环境中进行学习、二次开发或小规模业务运营。

项目目标不是单一图片上传站，而是一个可以继续扩展为商业化运营平台的综合系统：

- 图片上传、外链分发、Markdown / HTML / BBCode 链接生成
- 轻量个人网盘、文件夹管理、分享链接、提取码、有效期
- 用户注册登录、套餐购买、账户余额、订单与充值记录
- 阿里云 OSS、腾讯云 COS、七牛云 Kodo、本地测试存储统一接入
- 支付宝当面付、余额支付、人工充值审核和扩展支付通道结构
- 宝塔 API、SSH、HTTP 探针多方式服务器状态采集
- 公告中心、系统通知、平台帮助、教程内容管理
- 后台用户、套餐、订单、存储、文件、监控、公告、系统设置管理
- `/install` 可视化安装向导，适配宝塔面板部署

### 7. 服务器状态监控

BM TOOLS 的监控定位为运营者维护的服务器状态展示。普通用户前台只查看平台公开状态，不管理自己的服务器。

支持三种采集方式：

| 方式 | 说明 | 适合场景 |
|---|---|---|
| 宝塔 API | 通过宝塔面板 API 获取服务器状态 | 被监控服务器已安装宝塔 |
| SSH 采集 | 通过 SSH 执行只读命令采集状态 | 专业运维场景 |
| HTTP 探针 | 在被监控服务器放置 PHP 探针文件 | 不想安装 Composer 或 ssh2 扩展 |

采集指标包括：

- CPU 使用率
- 内存总量、已用、使用率
- 磁盘总量、已用、使用率
- 网络上传 / 下载速度
- 系统负载
- 在线时长
- Nginx 状态
- MySQL 状态
- PHP-FPM 状态
- 宝塔面板状态
- 采集错误和最近检查时间

监控能力：

- 后台添加服务器
- 测试连接
- 手动采集
- 定时采集
- 告警记录
- 邮件 / 站内通知
- 前台公开状态页 `/status`
- 监控数据新鲜度提示

定时任务示例：

```bash
php /www/wwwroot/你的站点/cron_monitor.php
```

建议在宝塔计划任务中每 1 分钟执行一次。

### 8. HTTP 探针监控

如果主站环境缺少 SSH 客户端能力，不想安装：

- Composer
- phpseclib/phpseclib
- PHP ssh2 扩展

可以使用 HTTP 探针方式。

项目内置探针文件：

```text
public/monitor-agent.php
```

部署步骤：

1. 将 `public/monitor-agent.php` 上传到被监控服务器的某个站点目录。
2. 修改探针文件顶部 Token。
3. 浏览器访问探针 URL，确认返回 JSON。
4. 后台监控管理中选择 `HTTP 探针`，填写探针 URL 和 Token。
5. 点击测试连接，成功后保存。

安全建议：

- 必须修改默认 Token
- 建议开启 HTTPS
- 不要向普通用户公开探针地址
- 可在 Nginx / 宝塔安全组中限制主站 IP 访问
- 探针只做状态读取，不执行外部传入命令

## 安装部署

### 环境要求

- PHP 7.4
- MySQL 5.7+
- PDO
- curl
- openssl
- fileinfo
- mbstring
- gd
- Apache 或 Nginx
- 推荐宝塔面板部署

SSH 直连监控可选依赖：

```bash
composer require phpseclib/phpseclib:^3.0
```

如果未安装 phpseclib，但 PHP 环境有 `ssh2` 扩展，系统可使用备用 SSH 通道。只使用宝塔 API 或 HTTP 探针监控时，不需要安装 Composer 依赖。

### 宝塔部署步骤

1. 在宝塔中新建 PHP 7.4 站点。
2. 创建 MySQL 5.7 数据库和独立数据库用户。
3. 上传项目代码到站点目录。
4. 推荐将网站运行目录设置为 `public`。
5. 如果无法设置运行目录，可使用根目录 `.htaccess` 或 Nginx 伪静态规则转发到 `public/index.php`。
6. 确认以下目录可写：

```text
config
storage
storage/logs
storage/cache
public/uploads
```

7. 访问安装向导：

```text
https://你的域名/install
```

8. 按步骤填写数据库信息、管理员账号和网站名称。
9. 安装完成后系统会生成：

```text
storage/install.lock
```

安装锁存在时，系统会阻止重复安装。

### Nginx 伪静态

项目提供：

```text
bt-nginx-rewrite.conf
```

可将其中规则复制到宝塔站点伪静态配置中。

### Apache

项目已提供：

```text
.htaccess
public/.htaccess
```
